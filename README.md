# API de Colaboradores

Projeto de API REST desenvolvida em Node.js com Express para gerenciamento de colaboradores.
Autor: Lucas tristão
Esse projeto foi feito por uma só pessoa.
---

##  Tecnologias utilizadas

- Node.js
- Express
- Axios
- File System (fs)
- JSON para persistência de dados

---


---

##  Funcionalidades da API

- Listar colaboradores (GET)
- Cadastrar colaboradores (POST)
- Integração com API externa (Axios)
- Persistência de dados em arquivo JSON
- Estrutura modular com rotas separadas

---

```bash
npm install
node app.js
